1. Accessibilità e SEO:
Aggiungi lang="it" già presente: ✅

Manca il tag <meta name="description">

2. Validazione timestamp:
Se a.timestamp non è un numero ma un oggetto Firebase Timestamp, il new Date() potrebbe non funzionare come previsto. Usa:

const dataOra = a.timestamp.toDate().toLocaleString("it-IT");
oppure controlla se è già in formato Date.

3. Fallback lato client se Firebase fallisce:
Puoi gestire errori così:

.onSnapshot(snapshot => { ... }, error => {
  lista.innerHTML = "<li>Errore nel caricamento annunci.</li>";
  console.error("Errore Firebase:", error);
});
